## Features

- **AI Nations**: 4-16 autonomous nations with unique personalities and diplomatic behaviors
- **Dynamic Diplomacy**: WAR, ALLIED, TRADE, TRIBUTARY relations that evolve based on strategic conditions
- **Territorial Control**: Procedurally generated terrain with mountains, rivers, and strategic nodes
- **Special Artifacts**: Research labs, weapon prototypes, fusion reactors with unlock chains
- **Superweapons**: Angels, EVA units, and experimental weapons with dramatic deployment sequences
- **Contested Capture**: Multi-nation competition for strategic locations
- **Spatial Audio**: Procedural sound effects with positional audio
- **TTS Announcements**: Dynamic event narration with nation-specific voices

## Controls

- **Click**: Start game (audio initialization)
- **Mouse Wheel**: Zoom in/out
- **Arrow Keys / WASD**: Pan camera
- **Space**: Pause/Resume
- **+/-**: Adjust game speed (0.05x to 5.0x)
- **L**: Download AI debug log
- **I**: Toggle info overlay

## Technology

- Pure HTML5/JavaScript - zero external dependencies
- Canvas 2D rendering with spatial partitioning
- Web Audio API for procedural sound synthesis
- Speech Synthesis API for event narration
- Single-file application (~700 KB)
